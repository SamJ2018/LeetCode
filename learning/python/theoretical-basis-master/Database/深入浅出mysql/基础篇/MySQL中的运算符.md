# MySQL中的运算符

> 用来连接表达式的项

- 算术运算符
- 比较运算符
- 逻辑运算符
- 位运算符

## 算术运算符

- 加、减、乘、除、模
- 小学数学的东西没什么好解释的

> 特殊：
>
> 当除数为0时，返回结果为NULL
>
> 对于模运算，除了可以用a%b表示还可用mod(a,b)表示

## 比较运算符

- 大于、小于、等于、不等于、大于等于、小于等于

  > 一些特殊
  >
  > 不等于 <> 或!=
  >
  > <=>也表示等于，但其即使操作的值为null也可以正确比较

- `between` 存在范围内，`in` 指定集合，`is  null` 判断是否为null

- `like` 通配符匹配 `regexp` 正则表达式匹配

  > regexp 使用格式 ` str regexp str_pat`
  >
  > like 使用格式 `a like `

## 逻辑运算符

- NOT或！ 逻辑非
- AND或&& 逻辑与
- OR或|| 逻辑或
- XOR 逻辑异或

## 位运算符

- & 位与
- | 位或
- ^ 位异或
- ~ 位取反
- \>> 位右移
- \<< 位左移

## 算符优先级

- 过多记不住，写的时候用()把需要优先的括起来
- 重要的自己选好认定
